# Ev

Interactive map application built with Felt SDK.

## Features

- Real-time data filtering with Felt SDK
- Interactive tooltips and feature details
- Responsive dark theme
- Stats dashboard

## View Your Map

https://felt.com/map/45JaGB9CDS9CW7mvDXifOC8A

## Run Locally

```bash
python3 -m http.server 8000
```

Then open: http://localhost:8000

## Built With

- [Felt](https://felt.com) - Modern web-based GIS platform
- [Felt JavaScript SDK](https://developers.felt.com/javascript-sdk/)
